## 五、会员服务条款

### 5.1 订阅说明

#### 5.1.1 订阅价格

- **月度会员**: 6.6元/月
- **自动续费**: 每月自动扣费
- **到期提醒**: 到期前7天会发送提醒

#### 5.1.2 订阅流程

1. 进入"数据&我" → "会员"
2. 点击"开通会员"按钮
3. 通过微信支付完成付款
4. 立即获得会员权限

### 5.2 取消订阅

#### 5.2.1 如何取消

在微信中：
1. 打开"微信" → "我" → "支付"
2. 点击右上角"..." → "扣费服务"
3. 找到"微习惯实验室" → 长按 → "关闭扣费服务"

#### 5.2.2 取消后

- 立即停止自动续费
- 当前计费周期继续有效，可继续使用到期
- 到期后降级为免费用户

### 5.3 退款政策

#### 5.3.1 可退款情况

根据微信支付规则，**特殊情况下**可申请退款：

- 📱 误操作导致的二次订阅（24小时内）
- 🐛 产品严重故障导致无法使用
- ❌ 未收到会员服务

#### 5.3.2 申请退款

1. 进入"数据&我" → 设置 → 意见反馈
2. 选择"退款申请" → 填写详细原因
3. 我们会在 7-14 个工作日内审核

**退款说明**：
- 退款将原路退至微信支付账户
- 取决于微信支付的政策和审核
- 我们无法保证所有退款申请都被批准

### 5.4 会员权益

#### 5.4.1 权益保证

会员享有以下权益：

| 权益 | 说明 |
|------|------|
| 无限习惯 | 可同时创建20个习惯（免费版限3个） |
| 完整21天 | 查看完整21天打卡日历（免费版仅7天） |
| 周月分析 | 7天和30天完成率趋势（免费版无） |
| 个性建议 | 基于数据的个性化改进建议 |
| 优先支持 | 优先处理问题反馈 |

#### 5.4.2 权益失效

会员到期或取消订阅后：
- ⏰ 高级功能立即不可用
- 📊 历史数据保留，免费版下仅可查看7天
- 🚫 无法新建第4个及以上习惯

### 5.5 续费说明

- **自动续费**: 每月自动从微信支付账户扣费
- **无缝服务**: 到期自动续费，服务不中断
